# PLAsTiCC Training Data

This directory contains the PLAsTiCC training and validation partitions (train + val), mirrored here for convenient download. These files are copies of the original PLAsTiCC dataset, not original work.

**Source**: Photometric LSST Astronomical Time-series Classification Challenge (PLAsTiCC, 2018)

**Contents**: Training set (7,066 objects) and validation set (1,564 objects) in Apache Parquet format, totalling approximately 35 MB.

**Usage**: Place these files in `software/data/raw/plasticc/data/` to enable the Demo and Benchmark tabs in AstroTransient.

**Full dataset**: The complete PLAsTiCC dataset (including the test partition, ~12 GB) is available separately and is not required for normal operation of this software.

# PLAsTiCC 训练数据

本目录包含 PLAsTiCC 训练与验证分区（train + val），为方便下载而镜像于此。这些文件是原始 PLAsTiCC 数据集的副本，非原创作品。

**来源**：Photometric LSST Astronomical Time-series Classification Challenge（PLAsTiCC，2018）

**内容**：训练集（7,066 个天体）与验证集（1,564 个天体），Apache Parquet 格式，总计约 35 MB。

**使用方式**：将文件放置于 `software/data/raw/plasticc/data/` 路径下，即可启用 AstroTransient 的 Demo 和 Benchmark 功能。

**完整数据集**：PLAsTiCC 完整数据集（含测试分区，约 12 GB）需另行获取，本软件正常运行无需此部分。
